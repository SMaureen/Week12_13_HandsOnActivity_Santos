# User Profile System - CodeIgniter 4

This project is a simple User Profile System developed using CodeIgniter 4. It allows users to upload profile images, store user information in a MySQL database, and display users with pagination.

## Features

* User profile image upload
* Image validation and storage
* Save user data to database
* Paginated user listing
* Delete user functionality

## Technologies Used

* PHP
* CodeIgniter 4
* MySQL
* XAMPP
